มี splash สีม่วงและการแสดงผลแบบลื่นขึ้น เพื่อลดความรู้สึกว่าหน้าจอขาวหรือกระตุกเวลาเปิด

ใช้ร่วมกับมือถือได้

คอมใช้ GitHub Dashboard ส่วนมือถือใช้ลิงก์เดิม ข้อมูลจะยังไปที่ Apps Script และ Google Sheet ชุดเดียวกัน

แนะนำการใช้งาน

  1. เปิดหน้านี้จาก GitHub Pages บนคอม
  2. กดปุ่ม เปิดระบบทะเบียนวิสัญญี เพื่อเข้า Web App หลักในแท็บใหม่
  3. ใช้งานบันทึกเคส ค้นหา หรือสถิติตามปกติในหน้า Apps Script
  4. ถ้าต้องการไอคอนบนคอม ให้ใช้เมนู Install app ของ Chrome หรือ Edge
หน้านี้เป็นหน้า Dashboard สำหรับคอมเท่านั้น และตั้งใจไม่ฝังแอปหลักแล้ว จึงไม่ติดปัญหา iframe
